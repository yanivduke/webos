# WebOS Code Editor Sample

This directory contains sample code files to demonstrate the **WebOS Code Editor** features.

## Features

- **Syntax Highlighting** for multiple languages
- **Line Numbers** with active line highlighting
- **Multiple Tabs** for working with multiple files
- **Find & Replace** with regex support
- **Auto-save** every 30 seconds
- **Keyboard Shortcuts** for common operations
- **Three Themes**: Amiga (retro), Dark, Light

## Sample Files

- `hello.js` - JavaScript example
- `styles.css` - CSS styling example
- `index.html` - HTML page example
- `config.json` - JSON configuration example
- `README.md` - This markdown file

## Keyboard Shortcuts

| Shortcut | Action |
|----------|--------|
| Ctrl+N | New File |
| Ctrl+O | Open File |
| Ctrl+S | Save File |
| Ctrl+F | Find |
| Ctrl+H | Find & Replace |
| Ctrl+G | Go to Line |
| Ctrl+/ | Toggle Comment |
| Ctrl+D | Duplicate Line |
| Ctrl+Shift+K | Delete Line |
